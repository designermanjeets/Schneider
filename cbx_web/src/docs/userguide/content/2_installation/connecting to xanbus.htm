Connecting the Conext Gateway to the Xanbus Network

To connect the Conext Gateway to the Xanbus network:
  1. Connect the Conext Gateway to the Xanbus network using daisy chain confguration.
  2. Xanbus components can be arranged in any order.
  3. Use a network terminator at both ends of the network. See Sample Xanbus network.

Notice

equipment damage

  • Do not connect a Xanbus cable plug into the Ethernet port on the Conext Gateway.
  • Connect only to Xanbus ports and use the network terminators to each end device in the daisy chain.

Failure to follow these instructions can result in equipment damage.
