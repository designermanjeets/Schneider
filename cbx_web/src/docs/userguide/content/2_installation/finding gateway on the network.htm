Finding the Conext Gateway on the Network

After the Conext Gatewayis wired into and powered up on a LAN, it will exist as a LAN device at an IP address. The following procedure describes how to find the Conext GatewayLAN address.

To find the Conext Gatewayon your network:
  1. Load the Conext GatewayCD (supplied) in the computer.
  2. If the computer in the Conext Gatewaysystem does not have a CD drive:
  3. On a computer with an optical drive, copy the Device Discovery Tool folder on the CD to a USB thumb drive.
  4. Temporarily disable all antivirus software including firewall protection software running in the background.
  5. Open the Device Discovery Tool folder either on the Conext GatewayCD or the USB thumb drive.
  6. Double-click ConextComBoxLocator.jar.
  7. This tool requires Java Runtime Environment (JRE). If you see a message about this, go to http://www.java.com/en/download/index.jsp to download and install JRE.
  8. In the list that appears, click the Web Address for the Conext Gateway
  9. (cb-XXXX where XXXX is a factory- assigned number).
  10. If a web browser opens up, you will see the Conext Gatewayuser interface Login window. Go to the next step.
  11. The default value of the Web Server Port is 80. For example, if you change this value to 8080, you must append the URL address with “:8080” at the end of the IP address (e.g. http://10.167.73.66:8080).
  12. If a web browser window does not open:
  13. Note the Web Address for the Conext Gateway.
  14. Open a web browser.
  15. Enter the Conext GatewayWeb Address in the URL field.
  16. Press Enter on the keyboard.
  17. The Conext Gatewayuser interface Login window appears.
  18. Enable all antivirus software including firewall protection software.
  19. To complete configuration of your Conext Gateway, go to the “Configuration” section.
