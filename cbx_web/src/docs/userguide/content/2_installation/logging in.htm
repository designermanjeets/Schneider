Logging in to the Conext Gateway Web Application

  1. Open a web browser.
  2. Type the web address of the Conext Gateway web application.
    IMPORTANT: The web address is a locally and privately assigned (LAN) device address that is also protected by a firewall.
  3. Select your User Name. Select Admin.
  4. Enter your Password. Initial password is Admin123.
  5. When prompted, change the initial password immediately to protect the device from unauthorized users and to enable changes to device settings. This is an important step to follow.

NOTE: To perform adminstrative functions such as a firmware update, set User Name to Admin. Settings are disabled until the initial password is changed.

  1. Change the Conext Gateway SSID password.
    1. Go to Setup > Network > Wifi Access Point Settings
    2. (Optional) Replace the current Wi-Fi network name under SSID field with an appropriate name, if so desired. Limit to 64 alphanumeric characters including symbols.
    3. Replace the current password under the Password field with 10 or more alphanumeric characters including symbols.
    4. Click Apply to save the new password and/or SSID.
