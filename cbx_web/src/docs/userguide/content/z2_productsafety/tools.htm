Required Tools

  • Complete set of electrical hand tools, including screwdriver sets with insulated handles (flat-head and Phillips)
  • Socket wrench set (5–25 mm)
  • Wrench set: open or box end (5–15 mm)
  • Metric hex key set, including an 8 mm (to open the inverter doors) and a 14 mm (for the anchor bolts)
  • Torx® driver set (T1–T10)
  • Calibrated torque wrench set (Nm or lb‑ft) with extensions, suitable for torque values ranging from 0.9–170 Nm (0.6–125 lb-ft)
  • Permanent marker pen
  • Anti-static wrist band with grounding wire
  • Voltage meter with minimum rating of 1500 VDC/CAT III
  • Voltage meter with minimum rating of 690 VAC/CAT IV

  • Test probes intended for and appropriately rated for use with the selected voltage
    meter(s)

    NOTE: Test probes with conductive probe tips that are 6.4 mm (0.25 in) in diameter and 136 mm (5.35 in) in length are required for probing through the inverter barriers, and an insulated handle length of 91.4 cm (3 ft) is required for probing the DC Recombiner section.

  • Schneider Electric-approved service tent (if required)
