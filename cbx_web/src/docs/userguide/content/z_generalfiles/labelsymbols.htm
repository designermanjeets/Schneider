Label Symbols

The following symbols appear on labels on or in the inverter.

Hearing damage hazard, wear hearing protection

Refer to the Installation or Operation instructions

Protective (grounding) conductor terminal

Functional grounding. Note that this relates to all of the shield ground connections in the Communications Wiring section.

Alternating current
