Changing the Password

What to expect when logging in for the first time

  • When logging in to the web application for the first time, either as Admin or User or Guest, the system will automatically ask you to change the password from the factory default Admin123 or User123 or Guest123, respectively.
Message box to change password

  • Click OK and follow the instructions to change the password.
Password screen

  • You may have to log in again after changing the initial password.
To change the password after the initial log in:
Change password setting

  1. Click Setup > Manage Passwords > Change Passwords.
  2. Select the User name type.
    • Admin
    • User
    • Guest
  3. Enter the Old password.
  4. Enter the New password.
  5. Enter the Confirm password.
  6. Click Apply. Your settings will be immediately applied.
