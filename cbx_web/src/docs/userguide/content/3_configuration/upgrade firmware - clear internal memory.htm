Clearing Internal Firmware Memory

To clear the Conext Gateway internal firmware memory:

Turn off the Conext Gateway by shutting it down and disconnecting all its power sources. Make sure the Power LED is off. Wait ten seconds before the next step.

Insert a pointed object such as a paper clip into the Reset pinhole and press the button while turning on the Conext Gateway by reconnecting to its power source.

Continue pressing (holding) the Reset pinhole button until all indicator LEDs flash quickly five times (approximately 10 seconds). Continue to hold the button until the LEDs flash slowly (approximately 5 seconds).

When the memory has been cleared, the Status LED is on and Power LED is blinking and a firmware upgrade must be done.
