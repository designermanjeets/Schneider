Master Slave Configuration

The Gateway can be set up in a master-slave configuration to monitor multiple separate Xanbus networks and supported Modbus devices, such as power meters. This configuration can be used for large multi-cluster systems as well as smaller systems when devices need to exist on separate ComBoxes.

Using a Master-Slave Configuration, multiple Xanbus networks on separate ComBoxes can be combined and monitored through the Master Gateway. When multiple ComBoxes have been added to the system, the settings of the slaves can be controlled and modified by the Master Gateway. Data is reported by the Slave Gateway to the Master Gateway. The Master Gateway is for data aggregation only and has no Xanbus or Modbus devices attached directly. RS-485 devices and principal devices such as XW+, AGS, SCP, etc. are placed on one of the slave ComBoxes.
