Using Copy Configuration

The copy configuration feature allows you to copy all device settings described in Table 3-1 from one device to another device of the same type on a common AC or DC connection.

To copy parameter settings from one device to another device of the same type:

Put the device on standby mode. See “To set a device in Standby mode:” on page 3–38.

Click System Devices to access the individual devices menu.

Click the device you want to configure.

Click in the left side menu.

You will see a list of Device Settings for that type of device.

Click to expand the device setting.

From the drop-down box, select the device to which you want to copy the settings.

Click to copy the settings.
