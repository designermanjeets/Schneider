Upgrading Device Firmware

It is possible to upgrade a device's firmware through the Conext Gateway web application.

Notice

risk of equipment damage

  • Consult the device’s Owner’s Guide when changing the device’s settings.

  • Make sure the device connected on the Xanbus or Modbus network is in Standby mode before changing settings.

  • Do not change any settings unless you are familiar with the device.

Failure to follow these instructions can result in equipment damage.

 Sample Device Overview screen - List display

To obtain the firmware package:
  1. From a laptop computer, open a web browser.
  2. Go to http://solar.schneider-electric.com and download the latest Conext Gateway firmware version (contained in a ZIP file).
  3. Navigate to Products > Battery Based Inverters > Accessories > Conext Gateway
  4. Click DOWNLOADS > Firmware > 865-0326 VX.X BNX
  5. Pick the firmware version (V) with a higher number and also a higher build number (BN). The ZIP file is approximately 35 MB.

NOTE: Pick the firmware version (V) with a higher number and also a higher build number (BN). The ZIP file is approximately 35 MB.

  1. Extract the .epkg file from the ZIP file into a folder on the loca drive.
To upgrade a device firmware:
  1. In the main menu bar, click Devices.
  2. Put the device on Standby first.
    1. Click a device you want to put on Standby.
    2. Click Configuration > Controls.
      Note that Controls may be called differently such as AGS Control.
    3. Search for the control setting Operating Mode.
    4. Select Standby.
    5. Click Apply.
  1. Click Firmware 
  2. Click the Upgrade button.
  3. Follow the step by step upgrade instructions.
  4. To upgrade all similar devices in the network, click the Upgrade All button.
  5. Follow the step by step upgrade instructions.
