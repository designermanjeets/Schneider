Changing Conext Gateway Settings

The procedures for configuring the other Conext Gateway Settings are described in the following sections.

To change Conext Gateway settings:
  1. In the main menu bar, click Setup > Configuration.
    The Configuration settings appear in the main display area.
  2. Click any of the following settings.
    1. Plant setup - see Plant Setup
    2. Time Setup - see Time Setup
    3. Import & export settings - see Import and Export Settings
    4. Units - see Units
    5. Modbus settings - see Modbus settings
    6. Restart Conext Gateway - see Restart Conext Gateway
    7. Install package - see Install Package
  3. With only a few exceptions, you may save a setting by clicking Apply or ignoring changes by clicking Cancel.
    That portion of the window expands to display the change options for that setting. The size and content of the expanded window varies depending on the setting type.
